Outil applicateur de couleur

(1) Cliquez sur une forme avec le bouton gauche pour appliquer la couleur primaire.

(2) Cliquez sur une forme avec le bouton droit pour appliquer la couleur secondaire.

Outil pipette

(1) Cliquez sur le canevas avec le bouton gauche pour remplacer la couleur primaire.

(2) Cliquez sur le canevas avec le bouton droit pour remplacer la couleur secondaire.
